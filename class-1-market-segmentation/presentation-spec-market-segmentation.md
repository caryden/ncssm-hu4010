## Key Visualizations to Build

### 1. Market Segmentation Matrix (Slide 13)
```
Structure:
- Animated table/grid
- 6-8 rows (segments)
- 5 columns (criteria) + total column
- Heat map coloring (1=red, 5=green)

Animation:
1. Grid structure appears (0.5s)
2. Segment names type in row by row (2s)
3. Column headers appear (0.5s)
4. Scores fill in cell by cell (3s)
5. Total column calculates (0.5s)
6. Top segment row highlights (0.5s)

Technical:
- D3 table generation
- Color scale: d3.interpolateRdYlGn
- Transition: staggered delays
```

### 2. Beachhead Expansion Diagram (Slide 23)
```
Structure:
- Central node (beachhead) - larger, pulsing
- Surrounding nodes (adjacent markets) - smaller
- Connecting lines with directional arrows
- Phase labels (1, 2, 3)

Animation:
1. Beachhead appears center with glow (1s)
2. First ring of adjacent markets fades in (1s)
3. Connection lines draw from beachhead outward (1.5s)
4. Second ring appears (1s)
5. "Phase 1, 2, 3" labels appear (1s)

Technical:
- D3 force-directed layout (fixed center)
- SVG glow filter on central node
- Curved paths for connections
```

### 3. Facebook Timeline (Slide 27)
```
Structure:
- Horizontal timeline
- 5 milestone nodes
- Growing circle representing user base
- Date labels below

Animation:
1. Timeline line draws left to right
2. First node appears with small circle
3. Each subsequent node: larger circle
4. Final node: largest, gradient fill

Visual metaphor:
- Circle size = user base
- Color intensity increases
```

### 4. Funnel Visualization (Slide 7)
```
Structure:
- Wide funnel at top
- 4 constraint bands narrowing
- Specific audience at bottom

Animation:
1. Wide opening appears
2. Each constraint "cuts" the funnel narrower
3. Final specific audience shown at bottom
4. "1M users in 2 months" result appears

Technical:
- SVG path animation
- Each band is a different shade of theme color
```
