## Key Visualizations to Build

### 1. The 24-Step Wheel (Slide 16)
**Critical visualization - appears in multiple classes**

```
Structure:
- SVG circle divided into 6 arc sections
- Each arc colored by theme
- 24 numbered nodes around circumference
- Labels for each theme in center or around edge

Animation sequence:
1. Center circle appears
2. 6 arcs draw in clockwise (1.5s total, staggered)
3. Theme labels fade in
4. Step numbers appear in sequence around wheel
5. Current session's steps pulse/glow

Technical notes:
- Use D3 for arc generation
- Store step data in JSON for reuse
- Create function to highlight specific steps (reused across classes)
```

### 2. Team Size Success Chart (Slide 12)
```
Structure:
- Horizontal bar chart
- 3 bars: Solo, 2-person, 3-person teams
- Values show relative success rate

Animation:
- Bars grow from left to right
- Staggered by 300ms
- Final bar has glow effect (winner)

Data:
- Solo: 1x (baseline)
- 2-person: 1.5x
- 3-person: 2x
```

### 3. Course Timeline (Slide 21)
```
Structure:
- Horizontal timeline
- Nodes for each session date
- Color coding for session types
- Spring break gap shown

Animation:
- Line draws from left to right
- Nodes appear as line reaches them
- Labels fade in after nodes

Visual markers:
- Tonight (Feb 2): "You are here" indicator
- May 4/11: Final pitches (larger nodes)
```
