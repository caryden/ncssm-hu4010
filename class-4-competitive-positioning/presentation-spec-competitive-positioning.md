### Core Section (Slides 14-22)

#### Slide 14: Section Divider
- **Type:** `slide-statement`
- **Content:**
  - Statement: "Step 10: Define Your Core"
- **Visual:** DE wheel with Step 10 glowing

#### Slide 15: What Is Core?
- **Type:** `slide-two-part`
- **Content:**
  - Primary: "Your Core"
  - Secondary: "The ONE thing you do better than anyone else that is ALSO hard to copy."
- **Animation:** Definition reveals with emphasis on "hard to copy"
- **Color:** Strong accent on key words

#### Slide 16: The Core Test
- **Type:** `slide-visual`
- **Content:**
  - Title: "The VRIO Test"
- **Visual:** Four-quadrant or checklist:
  - **V**aluable: Does the customer care?
  - **R**are: Is it uncommon?
  - **I**mitable: Is it hard to copy?
  - **O**rganized: Can you leverage it?
- **Animation:** Each criterion appears with question
- **All must be YES for true core**

#### Slide 17: Types of Core
- **Type:** `slide-visual`
- **Content:**
  - Title: "What Can Be Core?"
- **Visual:** Grid showing 8 types:
  1. Network Effects (more users = better)
  2. Proprietary Tech (unique IP)
  3. Cost Advantage (structural, not margins)
  4. Customer Lock-in (switching costs)
  5. Unique Expertise (specialized knowledge)
  6. Access/Relationships (exclusive partnerships)
  7. Brand (identity and loyalty)
  8. Data (proprietary information)
- **Animation:** Each type appears with icon and example
- **Color:** Each type has subtle color coding

#### Slide 18: Core vs Features
- **Type:** `slide-comparison`
- **Content:**
  - Left (Features - NOT Core):
    - "Fast delivery"
    - "Easy to use"
    - "Great customer service"
    - "Best price"
  - Right (Core - Defensible):
    - "Proprietary logistics network"
    - "Deep UX research methodology"
    - "Community-driven support model"
    - "Structural cost advantage"
- **Animation:** Left side with X marks, right side with check marks
- **Key insight:** "Features can be copied in 6 months. Core takes years."

#### Slide 19: Network Effects - The Holy Grail
- **Type:** `slide-visual`
- **Content:**
  - Title: "The Flywheel Effect"
- **Visual:** D3 animated flywheel:
  - More users → Better data → Better product → More users
  - Arrows connecting each stage in a loop
  - Loop spins faster over time
- **Animation:**
  1. First node appears
  2. Arrow to next node
  3. Complete cycle
  4. Flywheel starts spinning slowly, accelerates
- **Examples:** Uber, Google, Facebook

#### Slide 20: Finding Your Core - Questions
- **Type:** `slide-list`
- **Content:**
  - Title: "Ask Yourself"
  - List:
    - What do you know that others don't?
    - Who do you have access to that others don't?
    - What can you do that others can't (easily)?
    - What gets better the more you do it?
- **Animation:** Each question appears thoughtfully
- **Icon:** Question/search icon

#### Slide 21: Core Statement Template
- **Type:** `slide-visual`
- **Content:**
  - Title: "Write Your Core Statement"
- **Visual:** Fill-in template:
  ```
  Our core is [specific advantage]
  because [why it's rare/hard to copy]
  which allows us to [unique capability].
  ```
- **Example:**
  ```
  Our core is our network of verified restaurant reviewers
  because they've been built over 5 years of relationship building
  which allows us to provide reviews that competitors can't match.
  ```
- **Animation:** Template → example fills in

#### Slide 22: The Hard Truth
- **Type:** `slide-statement`
- **Content:**
  - Statement: "Most startups don't have a core yet. That's okay."
- **Subtext:** "Your job is to build one over time."

## Key Visualizations to Build

### 1. Network Diagram (Slide 10)
```
Structure:
- Force-directed layout
- You at center (larger node)
- First-degree connections (medium nodes)
- Second-degree connections (smaller nodes)
- Target customers highlighted (color/glow)

Animation:
1. Center node appears
2. First-degree nodes spawn and connect
3. Second-degree nodes spawn from first
4. Some outer nodes highlight as "potential customers"

Technical:
- D3 force simulation
- Node collision detection
- Highlight animation on target nodes
- Connection lines with subtle animation
```

### 2. Flywheel Animation (Slide 19)
```
Structure:
- Circular arrangement of 4 stages
- Curved arrows connecting stages
- Labels for each stage
- Center shows "flywheel" effect

Animation:
1. First stage appears
2. Arrow draws to second stage
3. Continue around circle
4. Once complete, all arrows begin rotating
5. Rotation accelerates over 5 seconds

Technical:
- SVG path animation for arrows
- CSS rotation animation for flywheel
- Easing: starts slow, speeds up
- Labels stay readable (don't rotate)
```

### 3. 2x2 Competitive Matrix (Slides 26-28)
```
Structure:
- X and Y axes with labels
- Grid lines (quadrant dividers)
- Plotted points with company logos/names
- Your position highlighted

Animation:
1. Axes draw (1s)
2. Axis labels fade in (0.5s)
3. Grid lines appear (0.5s)
4. Competitors plot one by one (staggered)
5. Your position appears last with emphasis

Technical:
- SVG for axes and grid
- Positioned icons/labels for competitors
- Highlight glow for your position
- Transition for adding new points
```

### 4. Core vs Features Comparison (Slide 18)
```
Structure:
- Two columns
- Left: Features with X marks
- Right: Core with check marks
- Visual transformation from left to right

Animation:
1. Feature appears
2. X mark animates
3. Corresponding core alternative appears
4. Check mark animates
5. Repeat for each row

Visual:
- Features slightly faded/crossed out
- Core statements bold and clear
- Transformation arrows optional
```

### 5. Apple vs Dell Timeline (Slides 4-5)
```
Structure:
- Timeline from 2005 to 2024
- Two company tracks (Dell, Apple)
- Key milestones and metrics
- Divergence visualization

Animation:
1. 2005: Both companies at similar scale
2. Key events plot (iPhone, etc.)
3. Lines diverge dramatically
4. 2024: Final positions revealed

Technical:
- D3 line chart with two series
- Annotations for key events
- Dramatic divergence effect
- Market cap numbers as data labels
```
