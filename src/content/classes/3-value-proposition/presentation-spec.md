### Opening Section (Slides 1-6)

#### Slide 1: Title Slide

- **Content:**
  - Title: "What Can You Do For Them?"
  - Subtitle: "Customer Journey & Value Proposition"
  - Session: "Class 3 | Steps 6, 7, 8"
- **Visual:** DE wheel with Steps 6-8 highlighted, Theme 2 purple accent

#### Slide 2: The Shift
- **Type:** `slide-two-part`
- **Content:**
  - Primary: "Last week: WHO is your customer"
  - Secondary: "This week: WHAT can you do for them"
- **Animation:** First line appears, then second with emphasis
- **Visual:** Arrow from "Who?" to "What?"

#### Slide 3: The Trap
- **Type:** `slide-statement`
- **Content:**
  - Statement: "Most founders talk about what their product DOES."
- **Animation:** Standard statement entrance

#### Slide 4: The Fix
- **Type:** `slide-statement`
- **Content:**
  - Statement: "Customers care about what it does FOR THEM."
- **Animation:** "FOR THEM" emphasizes with color shift
- **Color:** Accent purple on key phrase

#### Slide 5: The Uber Example - Before
- **Type:** `slide-visual`
- **Content:**
  - Title: "Getting a Taxi: Before Uber"
- **Visual:** Horizontal journey with pain points:
  1. Call taxi company (phone, wait on hold)
  2. Give address (miscommunication risk)
  3. Wait outside (unknown arrival time)
  4. Ride (hope meter is honest)
  5. Pay (fumble for cash, calculate tip)
  6. Receipt (maybe, if you remember)
- **Animation:** Steps appear left to right, each with red pain indicator
- **Emotion indicators:** Frustrated face icons at each step

#### Slide 6: The Uber Example - After
- **Type:** `slide-visual`
- **Content:**
  - Title: "Getting a Ride: With Uber"
- **Visual:** Same journey, transformed:
  1. Open app (see cars nearby)
  2. Enter destination (confirm price upfront)
  3. Track arrival (real-time ETA)
  4. Ride (follow along on map)
  5. Auto-pay (no fumbling)
  6. Receipt (automatic email)
- **Animation:** Steps transform from red to green
- **Emotion indicators:** Happy face icons replace frustrated ones

### Full Life Cycle Section (Slides 7-14)

#### Slide 7: Section Divider
- **Type:** `slide-statement`
- **Content:**
  - Statement: "Step 6: Full Life Cycle Use Case"
- **Visual:** DE wheel with Step 6 glowing

#### Slide 8: Why Map the Journey?
- **Type:** `slide-list`
- **Content:**
  - Title: "Your Product Doesn't Exist in Isolation"
  - List:
    - Customers experience you in context of their whole day
    - Every touchpoint is an opportunity (or failure point)
    - Understanding the journey reveals where to add value
    - Competitors might beat you in phases you forgot about
- **Animation:** Staggered list appearance
- **Icon:** Map/route icon

#### Slide 9: The Life Cycle Phases
- **Type:** `slide-visual`
- **Content:**
  - Title: "8 Phases of the Customer Journey"
- **Visual:** Circular or linear flow showing:
  1. Awareness (lightbulb)
  2. Consideration (scale/compare)
  3. Acquisition (shopping cart)
  4. Onboarding (rocket/launch)
  5. Usage (repeat/loop)
  6. Support (help/lifebuoy)
  7. Renewal (refresh)
  8. Churn (exit door)
- **Animation:** Each phase appears with icon, connecting arrows draw between
- **Color:** Gradient from awareness (light) to churn (dark)

#### Slide 10: Phase Deep Dive - Awareness
- **Type:** `slide-two-part`
- **Content:**
  - Primary: "Awareness"
  - Secondary: "How do they discover you exist?"
  - Questions:
    - What triggers the search?
    - Where do they look?
    - What do they search for?
- **Animation:** Questions appear as list
- **Icon:** Lightbulb

#### Slide 11: Phase Deep Dive - Consideration
- **Type:** `slide-two-part`
- **Content:**
  - Primary: "Consideration"
  - Secondary: "How do they evaluate options?"
  - Questions:
    - What alternatives do they compare?
    - What criteria matter most?
    - Who influences their decision?
- **Icon:** Scale/balance

#### Slide 12: Phase Deep Dive - Usage & Beyond
- **Type:** `slide-visual`
- **Content:**
  - Title: "The Core Experience & What Follows"
- **Visual:** Three connected circles:
  - Usage: The main job they hire you for
  - Support: When things go wrong
  - Renewal/Churn: What keeps them (or loses them)
- **Animation:** Circles appear and connect
- **Key point:** "Most founders focus only on Usage—don't make that mistake"

#### Slide 13: Journey Map Template
- **Type:** `slide-visual`
- **Content:**
  - Title: "Your Tool: The Journey Map"
- **Visual:** Template showing:
  - Horizontal timeline with phases
  - Row for "Actions" (what they do)
  - Row for "Thoughts" (what they think)
  - Row for "Feelings" (emotional state)
  - Row for "Pain Points" (opportunities)
- **Animation:** Template builds row by row
- **Interactive feel:** Ready to fill in

#### Slide 14: Journey Map Example
- **Type:** `slide-visual`
- **Content:**
  - Title: "Example: Student Finding Study Help"
- **Visual:** Completed journey map for a tutoring app:
  - Actions: Struggles with homework → Searches online → Compares apps → Downloads → First session → Regular use → Finals week surge
  - Feelings: Emoji progression (stressed → curious → hopeful → excited → confident)
  - Pain points: "Don't know what to search" / "Hard to compare options" / "Nervous first time"
- **Animation:** Rows fill in to demonstrate the process

### Value Quantification Section (Slides 20-28)

#### Slide 20: Section Divider
- **Type:** `slide-statement`
- **Content:**
  - Statement: "Step 8: Quantify the Value Proposition"
- **Visual:** DE wheel with Step 8 glowing

#### Slide 21: Why Quantify?
- **Type:** `slide-quote`
- **Content:**
  - Quote: "If you can't put a number on your value, you can't price your product."
  - Attribution: "Pricing truth"
- **Animation:** Quote fades in
- **Follow-up:** "Customers need numbers to justify decisions"

#### Slide 22: The As-Is / To-Be Framework
- **Type:** `slide-visual`
- **Content:**
  - Title: "The Value Equation"
- **Visual:** Animated comparison:
  - Left: "As-Is" (Without You) - red/negative shading
  - Right: "To-Be" (With You) - green/positive shading
  - Arrow between showing transformation
  - "Value = The Difference"
- **Animation:**
  1. As-Is state appears (pain indicators)
  2. To-Be state appears (improvement indicators)
  3. Arrow draws between
  4. "Value" calculation appears

#### Slide 23: Quantification Categories
- **Type:** `slide-visual`
- **Content:**
  - Title: "Types of Value"
- **Visual:** Four quadrants:
  1. **Time** (clock icon): Hours saved
  2. **Money** (dollar icon): Costs reduced
  3. **Risk** (shield icon): Problems prevented
  4. **Emotional** (heart icon): Peace of mind
- **Animation:** Each quadrant fills in with examples
- **Note:** "Time and Money are easiest to quantify. Don't ignore Risk and Emotional."

#### Slide 24: Time Value Calculation
- **Type:** `slide-visual`
- **Content:**
  - Title: "Quantifying Time Saved"
- **Visual:** Animated calculation:
  - Hours saved per week: 4
  - × Hourly value: $50
  - = Weekly value: $200
  - × 52 weeks
  - = Annual value: $10,400
- **Animation:** Numbers multiply and accumulate
- **Emphasis:** "That's the value you create—not your price"

#### Slide 25: Money Value Calculation
- **Type:** `slide-visual`
- **Content:**
  - Title: "Quantifying Money Saved"
- **Visual:** Before/After comparison:
  - Before: $500/month current cost
  - After: $200/month with you
  - Value: $300/month saved
  - Annual: $3,600/year
- **Animation:** Numbers transform

#### Slide 26: The 10x Rule
- **Type:** `slide-statement`
- **Content:**
  - Statement: "Your product should deliver 10x its price."
- **Subtext:** "Charge $100/month? Deliver $1,000/month of value."
- **Visual:** 10x in large gradient text
- **Animation:** 10x scales in with impact

#### Slide 27: Value Statement Template
- **Type:** `slide-visual`
- **Content:**
  - Title: "Write Your Value Proposition"
- **Visual:** Fill-in-the-blank template:
  ```
  We help [persona]
  save [time/money amount]
  by [core benefit]
  instead of [current painful solution].
  ```
- **Example:**
  ```
  We help busy restaurant managers
  save 10 hours/week
  by automating reservation management
  instead of manually juggling phone calls and notebooks.
  ```
- **Animation:** Template appears, then example fills in

#### Slide 28: Where We Are
- **Type:** `slide-visual`
- **Content:**
  - Title: "Theme 2 Progress"
- **Visual:** DE wheel with Steps 6, 7, 8 checked
  - Steps 9, 10, 11 previewed (next week)
- **Animation:** Checkmarks appear

## Key Visualizations to Build

### 1. Uber Journey Transformation (Slides 5-6)
```
Structure:
- Horizontal timeline with 6 steps
- Before state (red/frustrated)
- After state (green/satisfied)

Animation:
Slide 5: Before journey builds step by step
Slide 6: Transformation effect - each step morphs from red to green

Technical:
- SVG icons for each step
- Color transition animation
- Emoji/face indicators for emotion
```

### 2. Life Cycle Phases Diagram (Slide 9)
```
Structure:
- 8 phases in circular or infinity-loop arrangement
- Icons for each phase
- Connecting arrows showing flow

Animation:
1. First phase appears (Awareness)
2. Each subsequent phase appears with connecting arrow
3. Loop completes (Renewal back to Awareness, Churn as exit)

Technical:
- D3 for layout
- SVG icons from Lucide
- Animated path for connections
```

### 3. As-Is/To-Be Value Comparison (Slide 22)
```
Structure:
- Split screen: Left (As-Is), Right (To-Be)
- Multiple dimensions compared
- Value delta highlighted

Animation:
1. As-Is column appears (red indicators)
2. To-Be column appears (green indicators)
3. Difference arrows draw between
4. "Value" sum calculates at bottom

Technical:
- CSS grid layout
- Counter animation for numbers
- Arrow SVG drawing animation
```

### 4. Value Calculation Animation (Slides 24-25)
```
Structure:
- Math expression with variables
- Step-by-step calculation
- Final result emphasized

Animation:
1. Variables appear (4 hours × $50)
2. Intermediate result (=$200/week)
3. Next calculation (×52 weeks)
4. Final result (=$10,400/year) with celebration effect

Technical:
- D3 number interpolation
- Currency formatting
- Highlight animation on final number
```

### 5. Journey Map Template (Slide 13)
```
Structure:
- Grid with phases as columns
- Rows: Actions, Thoughts, Feelings, Pain Points
- Empty cells ready to fill

Animation:
1. Column headers appear (phases)
2. Row labels appear
3. Grid lines draw
4. Sample content fills one column

Interactive feel:
- Looks like a worksheet being prepared
```
